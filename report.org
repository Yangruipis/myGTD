* Daily Review

#+BEGIN: clocktable :maxlevel 2 :scope agenda-with-archives :block today :fileskip0 t :indent t

| File | Headline         | Time   |
|------+------------------+--------|
|      | ALL *Total time* | *0:00* |
#+END:

* How to use

- ~C-c C-x C-i~ 插入一个计时
- ~C-c C-x C-o~ 结束一个计时
- ~C-u C-c C-x C-i~ 查看最近的计时项目
